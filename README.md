# videochat-client
Embeddeable web app for video chat helpdesk. To be used with the [videochat-api][api] server.

## Install

```
npm install
```

## Build bundle and launch local webserver
```
npm start
```

## Contributing

### Launch dev local webserver
```
npm start dev.app
```

[api]: https://github.com/fczuardi/videochat-api
